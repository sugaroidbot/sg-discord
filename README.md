sg-discord
==========

A thin discord wrapper built on top of Sugaroid Websocket implementation.

![Screenshot](./docs/img/screenshot.png)

Build 
-----
```bash
go build .
```

Run
----
```bash
export DISCORD_BOT_TOKEN="supersecrettoken" 
export SG_DS_WS_ENDPOINT="localhost:3000"
./sg-discord
```
